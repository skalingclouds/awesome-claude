## Overview

Clarid AI offers a hosted Model Context Protocol server for marketing compliance review aimed at community banks and credit unions. The server checks advertising and promotional materials against regulations including FDIC, NCUA, TILA, Reg DD, Reg Z, UDAAP, and Equal Housing requirements.

The service is registered in the [official MCP Registry](https://registry.modelcontextprotocol.io/) as `ai.clarid/compliance` with remote endpoint `https://mcp.clarid.ai/mcp`. Source references are published at [github.com/clarid-ai/compliance-checker](https://github.com/clarid-ai/compliance-checker).

## Features

- Marketing compliance checks for banking and lending advertising rules.
- Streamable HTTP transport for remote MCP clients.
- Structured compliance feedback suitable for marketing and compliance teams.
- Hosted service with no local CLI installation required.
- Complements Clarid's broader compliance platform for HMDA, fair lending, and exam readiness.

## Use Cases

- Review a new deposit account landing page before publication.
- Check email campaign copy for UDAAP and disclosure issues.
- Validate social posts promoting mortgage or HELOC products.
- Screen branch signage language for Equal Housing and member-share disclaimers.
- Pre-flight website hero copy during a rebranding project.

## Installation

### Claude (Connectors)

1. Add a custom connector under **Settings → Connectors**.
2. Enter the MCP URL: `https://mcp.clarid.ai/mcp`.
3. Enable the connector and submit marketing copy through chat.

### Claude Code

```bash
claude mcp add --transport http clarid-compliance https://mcp.clarid.ai/mcp
claude mcp list
```

### Other MCP clients

Add a remote HTTP connector pointing at `https://mcp.clarid.ai/mcp` with streamable HTTP support.

## Configuration

```json
{
  "mcpServers": {
    "clarid-compliance": {
      "url": "https://mcp.clarid.ai/mcp",
      "type": "http"
```

## Security

- Clarid output is advisory; have qualified compliance staff or counsel approve final materials.
- Do not publish regulated copy based solely on automated MCP feedback.
- Restrict connector access to users who may handle draft marketing content.

## Troubleshooting

### Connector timeout

Retry after a few seconds; hosted endpoints may cold-start on first request.

### Incomplete findings

Provide full campaign context, product type, and audience so the model can evaluate disclosures correctly.

### Ambiguous product claims

Include APR/APY figures, fee schedules, and membership eligibility details when reviewing lending or deposit promotions.

### Wrong regulation scope

Clarid Compliance focuses on marketing rules; use Clarid's separate HMDA validator MCP for LAR file checks.
